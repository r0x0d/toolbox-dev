# Agent

Config key: `environments.agent`

## What it installs

**DNF packages:**

- `goose` — AI agent for development workflows

**OpenCode:**

Installed via the official install script from [opencode.ai](https://opencode.ai):

```bash
curl -fsSL https://opencode.ai/install | bash
```

## Usage

```yaml
environments:
  agent: true
```
